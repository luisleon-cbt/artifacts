Modern
Network Stack.

Objective 1.8 starts with the verb "summarize" — not explain, not configure. The exam tests whether you recognize six modern technologies (SDN, SD-WAN, VXLAN, ZTA, SASE/SSE, IaC, IPv6) and can match each to the problem it solves. This is the most acronym-dense topic on the exam. The artifact maps each concept to its purpose, contrasts the old way vs the new way, drills scenarios, and runs you through rapid acronym recognition.

The Six Modern Topics
SDN/SD-WANSoftware-defined networking — central policy, transport-agnostic
VXLANLayer 2 over Layer 3 for data center interconnect
ZTAZero trust — never trust, always verify
SASE/SSECloud-delivered network + security
IaCInfrastructure as code — automated provisioning
IPv6128-bit addressing — replaces exhausted IPv4
EXAM PATTERN
Scenarios match a problem to one of these technologies. "Branch sites connect over multiple ISPs with policy from one console" → SD-WAN. "Encrypted tunnel between two data centers carrying Layer 2 frames" → VXLAN/DCI.
