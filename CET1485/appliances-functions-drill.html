Objective 1.2 asks you to compare and contrast networking appliances (physical or virtual devices), applications (services), and functions (capabilities). The exam tests whether you can pick the right one for a scenario — and whether you know which category each belongs to. Routers, switches, and firewalls are appliances. VPN, QoS, and TTL are functions, not boxes. CDN is an application. This artifact drills the distinction in every tab.

Three Categories
AppliancesBoxes that do a job — router, switch, firewall, IDS/IPS, load balancer, proxy, NAS, SAN, AP, controller
ApplicationsServices delivered over networks — CDN
FunctionsCapabilities or features — VPN, QoS, TTL
EXAM TRAP
"Which appliance encrypts site-to-site traffic?" — wrong premise. VPN is a function, not an appliance. A firewall or router provides the VPN function.

Appliances

Appliances are the boxes — physical hardware or virtual machines — that do a specific networking job. They exist as devices you can rack, license, configure, and replace.

From the 1.2 list

RouterForwards Layer 3 packets between networks
SwitchForwards Layer 2 frames between hosts on the same network
FirewallFilters traffic based on policy rules
IDS / IPSDetects (IDS) or blocks (IPS) attack signatures
Load BalancerDistributes traffic across multiple backend servers
ProxyIntermediary that forwards requests on behalf of clients
NASFile-level network storage (SMB/NFS)
SANBlock-level storage network (Fibre Channel / iSCSI)
APWireless access point — radio bridge to wired network
ControllerCentrally manages many APs (or other devices)
Functions & Applications

Functions are capabilities — they're performed by appliances, not the appliances themselves. Applications are services that ride on top of networks. Neither is a single physical box.

Functions (capabilities)

VPNEncrypted tunnel between sites or for remote users — provided by firewalls, routers, or dedicated VPN concentrators
QoSTraffic prioritization — performed by switches, routers, or firewalls to favor voice/video over file transfers
TTLTime-to-Live — IP header field decremented at each router hop; prevents packets looping forever

Applications (services on the network)

CDNContent Delivery Network — geographically distributed cache that serves content to users from the nearest edge node
★ The Most-Tested Trap

"Which appliance provides VPN?" is a trick. VPN is a function. Multiple appliance types can provide it (firewall, router, dedicated VPN concentrator). The exam wants you to recognize the difference between a device and a capability.
