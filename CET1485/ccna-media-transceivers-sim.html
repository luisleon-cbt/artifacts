PICK A PORT ·
Shared Media (Half-Duplex Hub)

One Collision Domain

All four stations share one collision domain. The hub repeats every signal out every port. Only one station can transmit at a time without a collision. Stations use CSMA/CD — listen first, transmit, detect collisions.

Always half-duplex. A station cannot send and receive simultaneously on shared media.

Point-to-Point (Full-Duplex Switch)

Four Collision Domains (one per port)

Each switch port is its own point-to-point link with its own collision domain. With full-duplex, each station has separate TX and RX pairs and can send and receive at the same time. No collisions possible on a full-duplex P2P link.

CSMA/CD is disabled on full-duplex interfaces. Modern Ethernet is essentially always P2P + full-duplex.

About this Simulation

An interactive CCNA exam prep simulation covering Cisco 200-301 v1.1 Objective 1.3 — compare physical interface and cabling types. Covers single-mode fiber, multimode fiber, copper, DAC, and connection types (shared media vs point-to-point).

Built as a simulation rather than a drill: turn the distance and speed knobs in the Media Lab, watch the transceiver list update as you change ports, transmit on shared vs P2P to see CSMA/CD behavior, and read show interfaces output to diagnose real cable scenarios.
