EXAM TRAP
Scalability ≠ Elasticity. Scalability is the ability to grow. Elasticity is the ability to grow AND shrink automatically based on demand.

LEGEND: You manage this Provider manages this
IaaS — Infrastructure as a Service
You get VMs, networks, and storage. You install the OS, runtime, applications. Examples: AWS EC2, Azure Virtual Machines, Google Compute Engine.
PaaS — Platform as a Service
You upload code; the provider runs the OS, runtime, and middleware. Examples: Heroku, Google App Engine, AWS Elastic Beanstalk, Azure App Service.
SaaS — Software as a Service
You just use the app in a browser. No code, no OS, no servers. Examples: Gmail, Salesforce, Microsoft 365, Dropbox, Slack.
