HOW THIS WORKS
As your message moves down the sending stack, each layer wraps the data with its own header (and the Data Link layer adds a trailer too). The receiver does the reverse: each layer reads and strips its header, then hands the payload up to the next layer. The PDU name changes at each layer — that change is exactly what shows up on the exam.

Host Layers (5–7)

Run on the endpoints — your laptop, your server. They handle what the user is doing: opening a webpage, encrypting a session, keeping a video call alive. The OS implements them in software.

Media Layers (1–4)

Move data between hosts. Layer 4 is the boundary: TCP/UDP runs on the OS, but the rest is delivered by routers, switches, NICs, and cabling. This is where your network engineer earns her paycheck.

Exam Tips

· The PDU name is the layer giveaway. Frame → L2. Packet → L3. Segment → L4.

· Devices map to layers by what they read. Switch reads MACs (L2). Router reads IPs (L3).

· Layer 4 is where TCP vs UDP lives — flow control = TCP only.
